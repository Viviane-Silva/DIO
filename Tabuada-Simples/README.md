# 🧮 Função de Tabuada com JavaScript

Este repositório contém uma função simples chamada `mostrarTabuada(numero)`, desenvolvida em **JavaScript**, que imprime no console a tabuada do número informado, de **1 a 10**, utilizando a estrutura de repetição `for`.

---

## 📘 Exemplo de Uso

```javascript
mostrarTabuada(5);
```
